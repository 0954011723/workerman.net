# 目录结构
```sh
.
├── applications  // workerman的应用程序，里每个目录代表一个应用程序
│
└── workerman     // workerman主程序
    ├── bin       // 启动停止脚本
    ├── Common    // 一些workerman内置的应用程序，如WebServer、监控相关应用(Monitor,FileMonitor)
    ├── conf      // workerman配置
    ├── Core      // workerman核心文件
    └── logs      // workerman日志
```

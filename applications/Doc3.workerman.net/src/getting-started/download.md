# 下载

## 建议通过github clone下载，以便更新

github地址：https://github.com/walkor/workerman

centos系统中运行以下命令

```ssh
yum install git
git clone https://github.com/walkor/workerman
```

debian/ubuntu系统（非root用户需要配合sudo）

```ssh
apt-get install git
git clone https://github.com/walkor/workerman
```
